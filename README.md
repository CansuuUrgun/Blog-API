# My Blog API

A simple RESTful API for managing blog posts. This API supports full CRUD operations (Create, Read, Update, Delete) for blog posts.

## Features

- List all blog posts
- Retrieve a single blog post
- Create a new blog post
- Update an existing blog post
- Delete a blog post

## Technologies

- Node.js
- Express.js
- JavaScript (ES6)
- Nodemon (for development)
